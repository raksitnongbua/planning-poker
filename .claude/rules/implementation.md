# Implementation Rules

## Agent Usage
- Use the Agent tool (subagent) for frontend tasks

## Research First
- Before using any library or framework API, check if your knowledge might be outdated
- Use `mcp__next-devtools__nextjs_docs` or `mcp__next-devtools__nextjs_index` to look up current Next.js APIs before implementing

## MCP Next
- Always use MCP Next DevTools (`mcp__next-devtools__*`) when working on Next.js features

## When in Doubt
- Ask the user if there is not enough information to proceed — do not guess or assume
